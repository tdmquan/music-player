Hướng dẫn dành cho Linux

Bước 1: Giải nén tệp. Sau khi tải tệp từ website về, bạn vào bên trong thư mục đã lưu và giải nén. Bước 2: Vào terminal. Ubuntu thông thường sẽ được tích hợp sẵn Python3. Bạn có thể xem phiên bản hiện tại bằng lệnh python3 -V.

Bước 3: Tải các thư viện cần thiết để chạy app. App nghe nhạc của nhóm cần 3 thư viện để hoạt động bao gồm: Tkinter, Pygame, Pillow. A. Tải thư viện Tkinter. Bạn có thể tải thư viện này bằng lệnh: sudo apt install python3-tk.

Sau khi tải xong. Bạn vào Python3 và import thư viện đồng thời xem phiên bản của thư viện Tkinter bằng các lệnh: import tkinter tkinter.TkVersion

B. Tải thư viện Pygame Bạn có thể tải thư viện này bằng lệnh: sudo apt install python3-pygame.

Sau khi tải xong, vào python3 để import Pygame đồng thời xem phiên bản của thư viện bằng lệnh: import pygame

C. Tải thư viện Pillow Các phiên bản gần đây của Ubuntu thường đã tích hợp sẵn thư viện Pillow. Nếu chưa có thư viện này, có thể tải nó bằng lệnh: sudo apt install python3-pil

Ngoài ra, bên trong file python còn cần import thêm ImageTk từ thư viện Pillow. Vì vậy để chắn chắn. Hãy dùng lệnh: sudo apt install python3-pil.imagetk Để tải về ImageTK phòng trường hợp thư viện không có.

Bạn có thể vào python3 và lần lượt nhập các lệnh: import PIL PIL.__version__ Để có thể xem phiên bản hiện tại của thư viện Pillow

Bước 4: Chạy app. Cài đặt xong 3 thư viện trên thì đã có thể sử dụng app. Dùng lệnh cd để vào đường dẫn chứa file .py và chạy nó bằng python3.
